# Contributing to PoultryMitra

Thank you for your interest in contributing to PoultryMitra!

## Development Setup

1. Fork the repository
2. Clone your fork
3. Install dependencies:
```bash
cd frontend && npm install
cd ../backend && npm install
```

## Making Changes

1. Create a feature branch:
```bash
git checkout -b feature/your-feature-name
```

2. Make your changes
3. Test thoroughly
4. Commit with clear messages:
```bash
git commit -m "Add: feature description"
```

## Code Standards

### Frontend
- Use TypeScript
- Follow React best practices
- Use functional components with hooks
- Add proper type definitions
- Write accessible UI components

### Backend
- Use TypeScript
- Follow serverless best practices
- Add error handling
- Include logging
- Write unit tests

## Pull Request Process

1. Update documentation
2. Add tests for new features
3. Ensure all tests pass
4. Update README if needed
5. Submit PR with clear description

## Code Review

All submissions require review. We'll provide feedback and work with you to get your contribution merged.

## Questions?

Open an issue for discussion before starting major changes.
